## 🎯 核心卖点

### 技术亮点
1. **基于认知科学的评估体系** — 不是游戏化，是真正的认知成长
2. **五维认知模型** — 内化/应用/创造/元认知/协作
3. **自动团队扫描** — 无需手动配置，自动感知团队变化
4. **三模式归一化** — 适配单用户/小团队/大团队
5. **纯文本可视化** — 极简主义，不浮夸

### 应用场景
- 个人知识管理
- 团队认知成长追踪
- 学习型组织评估
- Agent 协作系统

### 目标用户
- 知识工作者
- 研发团队
- 学习型组织
- AI Agent 开发者

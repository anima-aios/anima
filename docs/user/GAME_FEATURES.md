## 📈 等级系统

### EXP 获取方式

| 行为 | EXP | 说明 |
|------|-----|------|
| 写 episodic 记忆 | +1 | 日常记录 |
| 写 semantic 记忆 | +2 | 知识沉淀（双倍） |
| 搜索记忆 | +2 | 主动检索 |
| 完成每日任务 | +5~20 | 难度相关 |
| 完成全部任务 | +15 | 额外奖励 |
| 高质量记忆 (S 级) | +50% | 质量奖励 |

### 等级公式

```
level = max(1, int(exp ^ 0.28))
```

| 等级 | EXP 需求 | 说明 |
|------|----------|------|
| Lv.1-10 | 0 → ~5,000 | 新手期 |
| Lv.11-20 | ~5,000 → ~20,000 | 成长期 |
| Lv.21-50 | ~20,000 → ~100,000 | 成熟期 |
| Lv.51-100 | ~100,000 → ~14,000,000 | 大师期 |

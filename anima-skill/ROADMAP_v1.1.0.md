### P0 - 核心功能（必须完成）

#### 1. 成就徽章系统

**功能描述：**
- 10 个成就徽章
- 解锁条件检测
- 徽章展示卡片

**徽章列表：**

| 徽章 | 名称 | 解锁条件 | 奖励 |
|------|------|----------|------|
| 🌱 | 新手起步 | 达到 Lv.5 | +50 EXP |
| 📚 | 知识达人 | 写 100 条 semantic 记忆 | +100 EXP |
| 🔍 | 搜索专家 | 搜索记忆 100 次 | +50 EXP |
| 📝 | 多产作家 | 连续 7 天写记忆 | +70 EXP |
| 🏃 | 持续成长 | 连续 30 天登录 | +150 EXP |
| 🤝 | 团队贡献 | 分享知识 20 次 | +100 EXP |
| 🎯 | 任务达人 | 完成 100 个每日任务 | +200 EXP |
| 💎 | 质量追求 | 获得 50 次 S 级评价 | +150 EXP |
| 🏆 | 排行榜首 | 连续 7 天第 1 名 | +300 EXP |
| 👑 | 终身成就 | 达到 Lv.50 | +1000 EXP |

**API 设计：**
```python
# 获取成就列表
get_achievements(agent_name) → List[Achievement]

# 获取解锁进度
get_achievement_progress(agent_name, achievement_id) → Progress

# 解锁成就（自动检测）
unlock_achievement(agent_name, achievement_id) → bool

# 生成成就展示卡片
generate_achievement_card(agent_name) → str
```

**数据格式：**
```json
{
  "achievements": [
    {
      "id": "novice_starter",
      "name": "新手起步",
      "icon": "🌱",
      "description": "达到 Lv.5",
      "condition": "level >= 5",
      "reward": 50,
      "unlocked": true,
      "unlocked_at": "2026-03-21T10:00:00Z"
    }
  ]
}
```

#### 2. 升级庆祝动画

**功能描述：**
- 升级时触发庆祝效果
- ASCII 动画展示
- 升级贺词

**动画效果：**
```
╔══════════════════════════════════════════╗
║         🎉 恭喜升级！ 🎉                  ║
║                                          ║
║         Lv.10  →  Lv.11                  ║
║                                          ║
║  🌟 新徽章：资深贡献者                    ║
║  🎁 奖励：+100 EXP                       ║
║                                          ║
║  继续加油，下一个目标 Lv.12！             ║
╚══════════════════════════════════════════╝
```

**API 设计：**
```python
# 检测升级并生成庆祝
celebrate_level_up(old_level, new_level) → str

# 生成升级贺词
generate_level_up_message(level) → str
```

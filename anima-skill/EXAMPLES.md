## ⚠️ 常见问题

### Q: 为什么显示 "core 未安装"？

```
A: 首次使用需要安装 anima-core 核心系统

运行以下命令：
bash ~/.openclaw/workspace-shuheng/projects/anima/anima-skill/post-install.sh

或手动安装：
git clone https://github.com/anima-aios/anima.git /tmp/anima
bash /tmp/anima/install.sh
```

### Q: EXP 怎么获得？

```
A: EXP 获取方式：

| 行为           | EXP  | 说明          |
|----------------|------|---------------|
| 写 episodic    | +1   | 日常记录      |
| 写 semantic    | +2   | 知识沉淀 (双倍)|
| 搜索记忆       | +2   | 主动检索      |
| 完成每日任务   | +5~20| 难度相关      |
| 分享知识       | +5   | 团队贡献      |
| 写周报/复盘    | +5   | 元认知        |
```

### Q: 如何提升等级？

```
A: 等级公式：level = max(1, int(exp ^ 0.28))

快速升级建议：
1. 每天写 3-5 条 semantic 记忆 (+6~10 EXP)
2. 完成每日任务 (+15~30 EXP)
3. 每周写周报 (+5 EXP)
4. 参与团队协作 (+3~5 EXP/次)

预计：
- 第 1 天：Lv.3 (100 EXP)
- 第 1 周：Lv.6 (1000 EXP)
- 第 1 月：Lv.13 (1 万 EXP)
```

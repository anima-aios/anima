## 🔌 OpenClaw Plugin Positioning

### What is Anima-AIOS?

Anima-AIOS is an **OpenClaw Skill Plugin** that provides OpenClaw Agents with:

- 🎮 **Gamified Growth System** - Learn and work like playing a game
- 📊 **Five-Dimensional Cognitive Assessment** - Understanding/Application/Creation/Metacognition/Collaboration
- 📈 **Level & EXP System** - Cumulative growth, never decrease
- 🏆 **Team Leaderboard** - Healthy competition, stimulate enthusiasm
- 📝 **Intelligent Memory Management** - Auto-sync, three-layer protection

## 🐛 Problems Solved

### Pain Point 1: Invisible Growth ❌
**Problem:** Studied and worked for a long time, don't know how much progress made

**Anima Solution:**
- ✅ Five-dimensional cognitive scores, clearly see ability distribution
- ✅ Level progress bar, real-time feedback on growth progress
- ✅ EXP history records, review growth trajectory

---

### Pain Point 2: Lack of Motivation ❌
**Problem:** Know should learn, but can't get motivated

**Anima Solution:**
- ✅ Daily task system, small goals easy to complete
- ✅ Instant EXP rewards, positive feedback loop
- ✅ Team leaderboard, healthy competition incentive

---

### Pain Point 3: Knowledge Hard to Sediment ❌
**Problem:** Learned a lot, but scattered everywhere, hard to reuse

**Anima Solution:**
- ✅ Memory management system, structured knowledge storage
- ✅ Quality assessment mechanism, encourage high-quality content
- ✅ Team shared library, more efficient knowledge reuse

---

### Pain Point 4: Ability Hard to Assess ❌
**Problem:** Don't know which aspects are strong, which need improvement

**Anima Solution:**
- ✅ Five-dimensional cognitive model (Understanding/Application/Creation/Metacognition/Collaboration)
- ✅ Radar chart visualization, strengths and weaknesses at a glance
- ✅ Targeted suggestions, clear improvement direction

## 🚀 Quick Start

### Installation (OpenClaw Environment)

```bash
# Install via ClawHub
clawhub install anima

# Or manual installation
cd ~/.openclaw/workspace-shuheng/skills/
ln -s ../../projects/anima/anima-skill anima
```

### Usage

```
# View cognitive profile
"What is my cognitive profile?"

# View EXP
"How much EXP do I have?"

# View daily tasks
"What are today's tasks?"

# Write memory
"Remember: Completed XXX today"

# Search memory
"Search memories about Vega"
```

### Self-Check

```bash
# Check Anima status
anima doctor

# Auto-fix
anima doctor --fix --auto
```

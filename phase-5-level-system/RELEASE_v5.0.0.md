## 🔄 升级指南

### 现有用户（Memora v4.x）

**无需重新安装！**

```bash
# 现有功能完全兼容
# 数据无需迁移
# 继续使用即可
```

**建议：**
1. 了解新品牌名称
2. 关注 GitHub 仓库
3. 参与社区建设

### 新用户

**直接安装 Anima-AIOS：**

```bash
# 克隆仓库
git clone https://github.com/anima-aios/anima.git

# 安装
cd anima
bash install.sh

# 初始化等级
python3 core/session_scanner.py
```

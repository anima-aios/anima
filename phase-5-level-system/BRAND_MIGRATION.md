# 品牌升级指南

**Memora → Anima-AIOS**

**版本：** v5.0.0  
**日期：** 2026-03-21

---

## 🎯 升级原因

### 原名：Memora

**含义：** Memory（记忆）  
**定位：** 记忆系统  
**局限：**
- ❌ 只体现"记忆"，不体现"认知评估"
- ❌ 不体现"等级成长"
- ❌ 名字局限性大

### 新名：Anima-AIOS

**含义：**
- **Anima** - 拉丁语"灵魂"
- **AIOS** - AI Operating System（AI 操作系统）

**优势：**
- ✅ 国际化（全球开发者都能理解）
- ✅ 有深度（拉丁语"灵魂"）
- ✅ 易传播（10 个字母）
- ✅ 符合定位（认知评估系统）
- ✅ GitHub 无直接竞争

## 🚀 升级步骤

### 对于现有用户

#### 步骤 1：更新认知

```
Memora v4.0.5 → Anima-AIOS v5.0.0
```

#### 步骤 2：继续使用

```bash

# 数据无需迁移
```

#### 步骤 3：了解新特性

```
v5.0.0 新增：
- 品牌升级
- GitHub 仓库
- 国际化支持
```

### 对于新用户

#### 直接安装 Anima-AIOS

```bash
# 克隆仓库
git clone https://github.com/anima-aios/anima.git

# 安装
cd anima
bash install.sh

# 初始化等级
python3 core/session_scanner.py
```
